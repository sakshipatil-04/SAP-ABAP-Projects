# Student Management System - SAP ABAP

## Description
This project is developed using SAP ABAP.

The project performs Student Management operations using custom database tables and ABAP reports.

---

## Features
- Insert Student Record
- Search Student by ID
- Update Student Details
- Delete Student Record
- Display Student Data using ALV Report

---

## SAP Concepts Used
- DDIC
- Open SQL
- Internal Tables
- Work Areas
- PARAMETERS
- LOOP
- CRUD Operations
- ALV Reports
- Field Catalog

---

## Custom Table Used
ZSTUDENTS_1

---

## Fields Used
- STU_ID
- S_NAME
- S_COURSE
- S_MARKS

---

## Programs Included

| Program Name | Description |
|---|---|
| zstudent_insert.abap | Insert student records |
| zstudent_search.abap | Search student by ID |
| zstudent_update.abap | Update student details |
| zstudent_delete.abap | Delete student record |
| zstudent_alv.abap | Display ALV report |

---

## Project Output
The project successfully performs CRUD operations and displays student data using ALV reports.
