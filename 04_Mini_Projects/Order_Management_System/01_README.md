# Order Management System - SAP ABAP

## Description
This project is developed using SAP ABAP.

The project displays Order Header and Order Item details using Interactive ALV Reports with custom database tables.

---

## Features
- Display Order Header Data
- Display Order Item Data
- Interactive ALV Report
- Header to Item Navigation
- Custom Field Catalog
- Custom Tables

---

## SAP Concepts Used
- DDIC
- Open SQL
- Internal Tables
- Work Areas
- Field Catalog
- Interactive ALV
- Callback FORM
- REUSE_ALV_GRID_DISPLAY

---

## Custom Tables Used
- ZORDER_1
- ZORDER_ITEM_1

---

## Order Header Fields
### ZORDER_1
- ORD_NO
- ORD_DATE
- PAY_MODE
- TOTAL_AMO
- CURR

---

## Order Item Fields
### ZORDER_ITEM_1
- ORD_NO
- OIN
- ODESC
- ICOST

---

## Program Included

| Program Name | Description |
|---|---|
| zorder_alv_report.abap | Interactive ALV Report for Order Management |

---

## Functional Flow
1. Main ALV displays Order Header data.
2. User clicks on Order Number.
3. Second ALV displays corresponding Order Item details.
