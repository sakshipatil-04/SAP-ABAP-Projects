# Custom Field Catalog ALV Report

## Description
This project demonstrates an ALV Report using Custom Field Catalog in SAP ABAP.

The report fetches sales document data from the VBAK table and displays it in ALV format with customized column headings.

---

## SAP Concepts Used
- Internal Tables
- Custom Types
- Field Catalog
- Open SQL
- REUSE_ALV_LIST_DISPLAY
- ALV Reporting

---

## Table Used
- VBAK

---

## Fields Displayed
- VBELN
- ERDAT
- ERZET
- ERNAM
- AUDAT


## Features
- Custom Column Names
- Column Positioning
- ALV Output Display
- Structured Report Design


## Program Name
ZLIBRARY_ALV_REPORT_1


## Output
The report displays sales document information in ALV list format.
