Table Name: ZLIBRARY_BOOK

Description:
This table stores library book details along with product and employee information.

Fields:
- BOOK_ID – Book ID
- BOOK_NAME – Book Name
- AUTHOR_NAME1 – Author Name
- BOOK_PRICE – Book Price
- PUBLISH_YEAR – Publish Year

Advanced Concepts:
- Include Structure (Product Details)
- Append Structure
- Multiple Data Types (INT, CHAR, DATS)

Use Case:
Library management system with extended product and employee tracking.
