# Include Structures

## Overview

Include Structures were created to store reusable fields that can be included in multiple SAP Dictionary objects.

This approach helps reduce redundancy and improves maintainability.

---

## Include Structures Created

### ZSTR_AUFK_INC

Used in:

* ZXX_AUFK

Purpose:

Contains common Production Order Header fields.

---

### ZSTR_AFKO_INC

Used in:

* ZXX_AFKO

Purpose:

Contains common BOM-related fields.

---

### ZSTR_AFPO_INC

Used in:

* ZXX_AFPO

Purpose:

Contains common Production Order Item fields.

---

## Benefits

* Reusability of fields
* Reduced duplicate development
* Easy maintenance
* Consistent data structure design

---

## SAP Concept

Data Dictionary (DDIC) → Include Structure

Include structures allow common fields to be reused across multiple tables and structures without redefining them.
