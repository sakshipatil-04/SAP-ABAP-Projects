# Database View

## Overview

A Database View was created to combine Production Order Header, BOM, and Item data into a single logical view.

---

## View Created

### ZV_AUFK_AFKO_AFP

Tables Used:

* ZXX_AUFK
* ZXX_AFKO
* ZXX_AFPO

---

## Purpose

* Retrieve data from multiple tables
* Simplify report development
* Reduce complex joins in programs

---

## Benefits

* Faster reporting
* Simplified data access
* Centralized business data

---

## SAP Concept

Data Dictionary (DDIC) → Database View

Database Views combine fields from multiple related tables into a single view for reporting and data retrieval.
