
# Search Helps

## Overview

Search Helps were created to provide F4 help functionality for Production Planning Order Management data.

Search Helps improve user experience by allowing users to search and select values instead of entering them manually.

---

## Search Helps Created

### ZSH_AUFNR_ELE

Elementary Search Help

Purpose:

Provides search help for Production Order Number (AUFNR).

Features:

* Displays Production Order Numbers
* Allows quick selection of order records
* Reduces manual entry errors

---

### ZSH_AUFNR_COLL

Collective Search Help

Purpose:

Combines multiple elementary search helps into a single search help.

Features:

* Centralized search functionality
* Multiple search criteria
* Improved user navigation

---

### ZSH_AUFNR_STLNR

Search Help for BOM Number

Purpose:

Provides F4 help for Bill of Material (BOM) Numbers.

Features:

* Easy BOM selection
* Faster data entry
* Improved data accuracy

---
