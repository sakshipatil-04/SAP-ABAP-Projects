# Module Pool Programming - SAP ABAP

## Description

This folder contains Module Pool Programming examples developed in SAP ABAP.

Module Pool Programming is used to create dialog-based applications with custom screens, user interactions, push buttons, subscreens, and database operations.

---

## Concepts Covered

- Screen Painter (SE51)
- Module Pool Programming
- Dialog Programming
- PBO (Process Before Output)
- PAI (Process After Input)
- User Commands
- Push Buttons
- Database Access
- Custom Tables
- Tabstrip Control
- Subscreens
- Screen Navigation
