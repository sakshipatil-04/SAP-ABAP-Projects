# Classical Reports in SAP ABAP

## 1. ZCLASSICAL_REPORT_2_TABLE

### Description
This report displays data from custom order header and item tables using Classical Report formatting.

### Tables Used
- ZORDER_1
- ZORDER_ITEM_1

### Features
- Internal Tables
- LOOP Statements
- Classical Report Formatting
- Column Alignment
- TOP-OF-PAGE
- END-OF-PAGE
- Page Number Display

### Output
Displays:
- Order Number
- Order Date
- Payment Mode
- Item Number
- Item Cost
- Currency

## 2. ZCLASSICAL_REPORT_EKKO_EKPO

### Description
This report displays purchase document data using standard SAP tables.

### Tables Used
- EKKO (Purchase Header)
- EKPO (Purchase Item)

### Features
- Selection Screen Parameter
- Classical Report Layout
- Header Formatting
- Page Numbering
- Standard SAP Table Data

### Output
Displays:
- Purchase Document
- Company Code
- Document Type
- Order Type
- Created By
