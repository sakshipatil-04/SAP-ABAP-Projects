# Interactive Reports in SAP ABAP

This folder contains SAP ABAP Interactive Report programs created by me.

## Programs

### 1. ZINTRACTIVE_REPORT
Interactive report using:
- HIDE technique
- AT LINE-SELECTION
- TOP-OF-PAGE
- Secondary list display
- Header and item table display

## Tables Used
- ZORDER_1
- ZORDER_ITEM_1

## Features
- Displays Order Header data
- Click on an order to view Item details
- Classical interactive report format
- Formatted output using SY-VLINE and SY-ULINE

## Files
- `zintractive_report.abap` → ABAP source code
- `zintractive_report_output_1.png` → Primary list output
- `zintractive_report_output_2.png` → Secondary list output
